How Does A Garage Door Light Trigger Work?

A garage door light trigger uses a sensor to detect when the garage door is opened and closed. When the door is opened, the sensor activates the light switch, which turns the lights on. When the door is closed, the sensor sends a signal to the switch, which turns the lights off. It is a simple process that can be done quickly and easily.
