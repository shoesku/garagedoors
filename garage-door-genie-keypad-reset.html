How To Reset Your Garage Door Genie Keypad

Do you have a Genie garage door opener but can’t remember the code you set? Don’t worry, resetting your Genie garage door keypad is a simple process that only takes a few minutes. In this article, we’ll walk you through the steps you need to take to reset your Genie garage door keypad and get up and running again.

Tools You’ll Need

Before you can reset your Genie garage door keypad, there are a few tools that you’ll need. These include:

  • A flathead screwdriver
  • A Phillips head screwdriver
  • A ladder
  • A new Genie keypad
  • A pair of wire cutters

Step 1: Disconnect the Power Source

The first step you’ll need to take to reset your Genie garage door keypad is to disconnect the power source. Find the power cord connected to the keypad, and use a Phillips head screwdriver to loosen the screws and disconnect it. Make sure that the power is completely off before you proceed.

Step 2: Remove the Old Keypad

Once the power is disconnected, you’ll need to remove the old keypad. There will be two screws holding the keypad in place, and you’ll need to use a flathead screwdriver to remove them. Once the screws are removed, you should be able to easily slide the keypad off the wall.

Step 3: Install the New Keypad

Now you’ll need to install the new Genie keypad. Start by connecting the wires to the back of the keypad. You may need to use a pair of wire cutters to shorten the wires if they’re too long. Once the wires are connected, use the two screws to secure the keypad to the wall.

Step 4: Reconnect the Power Source

Now you’ll need to reconnect the power source. Make sure that the power is off before you do this. Use the Phillips head screwdriver to tighten the screws, then turn the power back on. This should activate the new keypad.

Step 5: Program the Keypad

The last step is to program the new keypad. This process will vary depending on the model of your Genie opener, so you’ll need to refer to the user manual for specific instructions. Most Genie openers allow you to program up to three different codes, so make sure you choose a code that you’ll remember.

Conclusion

Resetting your Genie garage door keypad is a simple task that only takes a few minutes. Just make sure that you have all the necessary tools, disconnect the power source, remove the old keypad, install the new keypad, reconnect the power source, and program the new keypad. Follow these steps and you’ll have your Genie opener up and running again in no time.
