Types of Garage Door Extension Springs

There are two main types of extension springs available for garage doors. The first type is a torsion spring, which is a type of spring that is wound tightly and then released when the door is opened. The second type is an extension spring, which is a type of spring that is attached to the door and then stretched out when the door is opened. Both types of springs provide benefits to homeowners, but torsion springs are typically more expensive than extension springs.
